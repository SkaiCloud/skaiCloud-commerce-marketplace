# Post-install checks

1. Open `/dashboard/storefront` with Marketplace access.
2. Create a draft product with a name, SKU, price, inventory, category, description, and two images.
3. Confirm the product remains absent from `/marketplace` while it is a draft.
4. Publish the product and confirm its card and detail view appear publicly.
5. Set inventory to zero and confirm the public status changes to out of stock.
6. Archive the product and confirm it is removed from the public catalog.
7. Disable and re-enable the module and confirm product records and images remain intact.
